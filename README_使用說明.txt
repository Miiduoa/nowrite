============================================================
 OS 手寫小抄產生器 — Windows 版使用說明
============================================================

這個套件可以把「文字檔」用指定字體（TekitouPoem.ttf）轉成
看起來像手寫的 PDF，所有參數（字體大小、行間距、各種擾動、
墨汁深度、塗改…）都能在圖形介面裡調整。

資料夾內容：
  app.py ............................ 圖形介面程式
  render_core.py .................... 手寫渲染核心
  TekitouPoem.ttf ................... 字體
  OS_Cheat_Sheet_handwriting_source.txt  預設要手寫的文字（可換成任何 .txt）
  build_exe.bat ..................... 一鍵編譯成獨立 EXE（推薦）
  run_app.bat ....................... 已裝 Python 者直接執行
  requirements.txt .................. 相依套件清單


────────────────────────────────────────────
 方法 A（推薦）：做成獨立 EXE，之後免裝 Python
────────────────────────────────────────────
 1. 雙擊  build_exe.bat
    - 若電腦沒有 Python，它會用 winget 幫你自動安裝；
      裝完請關掉視窗，再雙擊 build_exe.bat 一次。
    - 它會自動安裝套件並編譯。第一次約 1–3 分鐘。
 2. 完成後，程式在：  dist\OS_Handwriting_Generator.exe
 3. 以後直接雙擊那個 EXE 就能用，不需要 Python。
    ※ 請讓 EXE 與 TekitouPoem.ttf、.txt 放在同一個資料夾
      （build 已自動複製到 dist\ 內）。


────────────────────────────────────────────
 方法 B：已經有 Python，想直接用
────────────────────────────────────────────
 雙擊  run_app.bat  （會自動裝套件並開啟程式）
 或手動：
   pip install pillow numpy handright fonttools
   python app.py


────────────────────────────────────────────
 怎麼使用程式
────────────────────────────────────────────
 1. 「字體」「文字檔」「輸出 PDF」三個欄位確認或用「瀏覽」選擇。
    - 想手寫別的內容，就把文字另存成 .txt，在這裡選它即可。
 2. 選「紙張」(空白 / 橫線 / 方格) 與「墨色」(黑 / 藍筆 / 深灰)。
 3. 調整下方參數（預設已經是你給的設定）：

      字體大小=73      行間距=74
      上邊距=42        下邊距=43      左邊距=50   右邊距=50
      行間距擾動=3     字體大小擾動=3  字間距擾動=3
      筆畫橫向偏移=4   筆畫縱向偏移=4  筆畫旋轉偏移=0.05
      字間距=1
      塗改線長度標準差=2  塗改線角度標準差=2  塗改線寬度標準差=1
      塗改出現的幾率=0.005  塗改線寬度=6   墨汁深度標準差=28
      解析度 DPI=200   亂數種子=20260605（換數字可得到不同字跡）

 4. 按「生成 PDF」。完成後會自動開啟輸出資料夾，
    同時也會輸出每一頁的 PNG（handwritten_page_01.png …）。


────────────────────────────────────────────
 小提醒
────────────────────────────────────────────
 - 這是「手寫模擬」：表格/分欄會變成連續手寫文字行。
 - 這個日文字體缺少少數繁體字（例：查、產），程式會自動換成
   同字的日文寫法（査、産）以免出現空白方塊。
 - 想要不同字跡：改「亂數種子」即可。
 - 想換字體：把任何 .ttf 在「字體」欄選進來即可。
 - Windows SmartScreen 第一次可能提示未知發行者，
   選「更多資訊 → 仍要執行」即可（因為是你自己編譯的程式）。
